# 游戏大厅项目 - 环境部署说明

## 一、服务器环境要求

### 1. 操作系统
- Linux (推荐 Ubuntu 20.04/22.04 或 CentOS 7/8)

### 2. 必需软件
- **Java 21** (OpenJDK 21)
- **Node.js 20+** (用于前端构建)
- **npm** (Node包管理器)
- **SQLite3** (数据库)

### 3. 安装命令参考

#### Ubuntu/Debian
```bash
# 安装Java 21
apt update
apt install openjdk-21-jdk -y

# 安装Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install nodejs -y

# 安装SQLite
apt install sqlite3 -y
```

#### CentOS/RHEL
```bash
# 安装Java 21
yum install java-21-openjdk-devel -y

# 安装Node.js 20
curl -fsSL https://rpm.nodesource.com/setup_20.x | bash -
yum install nodejs -y

# 安装SQLite
yum install sqlite -y
```

## 二、项目部署步骤

### 1. 上传项目文件
将整个 `game-lobby` 目录上传到服务器，例如 `/root/game-lobby`

### 2. 安装前端依赖
```bash
cd /root/game-lobby/frontend
npm install
```

### 3. 构建前端
```bash
npm run build
```
构建产物将输出到 `frontend/target/` 目录

### 4. 构建后端
```bash
cd /root/game-lobby/backend
chmod +x build.sh start.sh stop.sh
./build.sh
```
构建产物将输出到 `backend/target/` 目录

### 5. 启动服务
```bash
./start.sh
```

### 6. 停止服务
```bash
./stop.sh
```

### 3. AI配置
配置文件位置：`backend/ai-config.properties`

| 配置项 | 值 | 说明 |
|--------|-----|------|
| ai.api.url | https://api.openai.com/v1 | API基础URL |
| ai.api.key | sk-your-api-key-here | API密钥（需替换） |
| ai.model | gpt-3.5-turbo | 模型名称 |
| ai.max.tokens | 2000 | 最大Token数 |
| ai.temperature | 0.7 | 温度参数 |
| ai.enabled | false | 是否启用AI |

**注意：** 使用AI功能需要：
1. 将 `ai.enabled` 改为 `true`
2. 将 `ai.api.key` 替换为有效的API密钥
3. 根据实际API服务调整 `ai.api.url` 和 `ai.model`

## 五、网络配置

### 1. 防火墙配置
需要开放以下端口：
- **80端口**：HTTP服务（必须）

#### Ubuntu UFW
```bash
ufw allow 80/tcp
ufw reload
```

#### CentOS Firewalld
```bash
firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --reload
```

### 2. 云服务器安全组
如果使用云服务器（阿里云、腾讯云、AWS等），需要在安全组中添加规则：
- 协议：TCP
- 端口：80
- 来源：0.0.0.0/0（或指定IP）

## 八、常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -tlnp | grep :80

# 停止占用进程
kill -9 <PID>
```

### 2. Java版本不对
```bash
# 检查Java版本
java -version

# 应显示 openjdk version "21.x.x"
```

### 3. 前端构建失败
```bash
# 清除依赖重新安装
cd frontend
rm -rf node_modules package-lock.json
npm install
npm run build
```

### 4. 数据库问题
数据库文件位于 `backend/target/game_lobby.db`，如需重置：
```bash
cd backend
# 备份原数据库
cp game_lobby.db game_lobby.db.bak
# 重新初始化（需要重新运行数据库初始化脚本）
```

1. **修改数据库密码**：生产环境建议修改 `database.properties` 中的密码

3. **定期备份**：定期备份 `game_lobby.db` 数据库文件
4. **日志监控**：定期检查 `server.log` 日志文件
